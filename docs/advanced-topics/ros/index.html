From FRC Team 624 - CRyptonite Robotics from Texas​

  • 2020 ROS discussion on Chief Delphi

Team 624 (CRyptonite Robotics) from Texas used a coprocessor from Up-Board along with WPIlib Java on the RoboRIO. They ran a ROS environment on Ubuntu 20.04 on the co-processor for custom vision tracking and autonomous path planning.

  • Github - LeonidasVarveropoulos/robot-frc: This is a ROS catkin workspace for a robot in frc
  • GitHub - LeonidasVarveropoulos/UnitySimulator-ROS: This is a Unity project that works as a simulator for the ROS FRC robot code hosted in the robot-frc repo.
